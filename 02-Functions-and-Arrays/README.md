# Functions and arrays
This folder focuses on organizing code into reusable blocks and managing collections of data.

### Projects:
- **Unit Convertor:** A C++ program that converts between units using functions and switch-case.
- **Palindrome Checker:** A C++ program that determines if a given string is a palindrome.
- **Binary Search Tool:** A C++ program that determines if a given number exists in an array using the Binary Search algorithm.
